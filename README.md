# Event Registration Portal

## 📌 Project Description
This is a web-based application that allows users to register for events easily.

## 🚀 Features
- User registration form
- Event details display
- Data storage
- Simple and user-friendly interface

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Node.js (if used)

## ▶️ How to Run
1. Download the project
2. Run `npm install`
3. Start server using `node app.js` (or your main file)
